# VNEngine
A portable visual novel engine in pure c with scripts in lua

[Docs](https://github.com/hibikidesu/vnengine/wiki/Lua-API)

## Dependencies
- sdl2
- sdl2_ttf
- sdl2_mixer
- sdl2_image
- zlib
- luajit
